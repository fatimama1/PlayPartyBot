# BoardBuddy

Telegram-бот для организации настольных игр и мини-игр для друзей.

## Функции
- Сбор компании друзей для настолок: бот уведомляет участников и спрашивает, придут ли они.
- Слот-машина с учётом баланса пользователя — мини-игра прямо в чате.
- Настроено с использованием Docker и Alembic для удобного деплоя и миграций базы данных.
